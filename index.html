Get started with TS Query

TS Query provides a lightweight, TypeScript-based library that mimics core jQuery functionality like DOM traversal, event handling, and manipulation using modern JavaScript under the hood.

It's type-safe, chainable, and dependency-free designed for performance and clarity in modern frontend projects.

  • DOM Manipulation

This section covers the primary set of functions designed to replicate jQuerys most commonly used features in a modern, type-safe environment.

$(selector)

Get elements matching the given selector. Works for any valid CSS selector.
